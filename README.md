# BETTER AUTH STARTER

Key features:

- Email & password authentication
- Email verification
- Reset password
- Github authentication
- Gooogle authentication (Not set up yet)
- Resend
- Email templates
- Prisma ORM
- TailwindCSS

### Prerequisites

### Cloning the repository

```bash
https://github.com/Angelleduck/better-authjs.git
```

### Install packages

```bash
npm i
```

### Setup .env file

```bash
# A secret key
BETTER_AUTH_SECRET=

# The URL of The App in prod or dev
BETTER_AUTH_URL=

#URL to connect to prisma
DATABASE_URL=

# Api key to send email
RESEND_KEY=

# Key provided by generated by github to auth with github
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
```

### Setup Prisma

```bash
npx prisma generate
```

### Start the app

```bash
npm run dev
```
